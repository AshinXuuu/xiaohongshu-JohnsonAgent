# 乔山销售助手 v3.2 更新公告

> 更新日期:2026-07-03
> 面向:全国 KOS / 一线业务 + 管理员
> 地址:https://johnson-xhs.xxcode.work/

这一版围绕**KOS 任务中心**和**小红书生成助手**做了一大批打磨,并完成了一次**账号安全升级**。下面按你们用得到的地方说明。

---

## 一、账号安全升级(无需操作,自动生效)

登录后系统会为你签发一个加密的登录凭证,后续所有操作都用它来确认身份——**别人再也无法冒用你的工号或管理员身份**。你唯一能感觉到的变化是:登录状态有效期 24 小时,过期后重新用工号登录一次即可。

---

## 二、KOS 任务中心

### 素材改为三分类,出图统一 3:4

- 建素材库后,上传按钮变成**三个**:主图 / 2合1(横版)/ 4合1(竖版),分开上传、分开计数。
- 成品图统一裁成小红书竖图 **3:4(1080×1440)**:主图裁一张;2合1 是两张横版上下拼;4合1 是四张竖版田字拼——**先把每张裁到精确尺寸再无缝拼接**,不会二次损失。
- 修复了**手机竖拍照片被转向**的问题(按方向自动转正)。
- 素材数量与 COS 桶**实时对齐**,管理员无需进后台核对。

### 领取页大改

- 顶部两张卡改为**本期**口径(周四发布 → 下周四回收):
  - 卡一「本期任务笔记(已完成 / 需完成)」;
  - 卡二「本期自发布笔记」。
- 新增「**自发布笔记**」板块:自己发的(非任务)小红书笔记,把链接贴进来登记,单独计数并进排行。
- **已领素材包**:每人 1 篇的任务,点开直接显示素材包;多篇任务才显示列表。
- 文案改为**标题 / 正文 / 标签 分栏,各自一键复制**。
- **回收链接**:只接受小红书链接(手机端 xhslink.com 或电脑端 xiaohongshu.com),而且会**自动从你粘贴的整段口令里提取纯链接**——修复了之前点链接跳到乱码页、无法打开的问题。
- 排行榜按「本期任务完成 + 本期自发布」排名。

### 管理后台

- KOS 数据里新增「**任务笔记链接**」列表,管理员可直接点开查看每条已发布的笔记。

---

## 三、小红书生成助手

### 文案

- 选完品牌 + 产品,页面上有**两个一键按钮**:一键生成文案 / 一键生成封面图。
- 文案每次生成**都不一样**(随机切换种草角度),整体都是种草向;**话题标签固定为同一批**(品牌 + 产品 + 通用标签),不再时有时无。

### 封面(重点升级)

- **封面仍需你上传一张产品照**,照片主体保持不变。
- 「**一键导入**」智能兜底:生成过文案就按文案内容出封面文字,没生成也能直接按产品资料出——不再要求必须先写文案。
- **换用更强的出图模型**,中文字更稳。
- **提示词全面重做**,更贴近小红书爆款封面:
  - 主标题 / 副标题 / 花字**三级层级**,一字不差、强制简体、不出现繁体和多余文字;
  - **多色分层 + 关键词高亮**(白+黄、亮绿描边、红黄撞色等),关键词或数字单独放大变色;
  - 超粗黑体 + 厚描边 + 立体感,照片上也清晰醒目;
  - 共 **18 套配色方案**(白黄经典 / 亮绿 / 白橙 / 撞色卡片 / 黑金 / 粉白 / 奶茶 / 蓝紫渐变 / 荧光撞色…),每次随机出 3 张;
  - **不再给照片加滤镜**,变化集中在文字的风格、颜色、字体和排列。

## 四、体验与性能

- 服务器改为**多线程并发**:多张图并行加载、别人领素材时也不再卡住你,页面明显更快;成品图带缓存,重复打开秒开。
- 修复小红书页刷新时**闪现历史登录页**的问题。
- 前端**去掉了所有技术模型名词**,界面更干净统一。
