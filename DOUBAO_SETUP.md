# 豆包 SeedEdit 接入手册

封面图生成走的是**火山引擎 Ark 平台**(volcengine.com),用的是字节跳动家的"豆包·图像编辑 3.0"模型。

预计耗时:**20 分钟**(注册 + 充值 + 拿 key + Vercel 配置)

---

## Step 1:注册火山引擎

打开 https://www.volcengine.com/,用手机号或微信注册账号。

注意:**只能用企业账号**做实名认证才能正常调 API,个人账号有时被限。你公司有营业执照就能开企业。

完成实名认证(企业实名 1-3 个工作日,加急 30 分钟内)。

---

## Step 2:开通方舟(Ark)+ 充值

1. 进 https://console.volcengine.com/ark/
2. 点 **开通服务**,同意协议
3. **充值 10-50 元**(豆包视觉模型单次调用约 ¥0.1-0.3,我们每次出 3 张 = ¥0.3-0.9。10 元够测试很多次)
4. 在 **模型管理** 里找到 **豆包·图像编辑 SeedEdit 3.0 i2i**(模型 ID:`doubao-seededit-3-0-i2i-250628`)
5. 点 **开通使用**,确认就行(免费开通,只在调用时按量扣费)

---

## Step 3:创建 API Key

1. 进 https://console.volcengine.com/ark/region:ark+cn-beijing/apiKey
2. 点 **创建 API Key**
3. 名字写 `xiaohongshu-agent`(随意)
4. **权限范围**勾选 **图像生成**(确保覆盖到 SeedEdit)
5. 创建后会出现一长串 key,以 `xxxxx-xxxxx-xxxxx` 形式,**只显示一次,马上复制下来**

---

## Step 4:把 Key 加到 Vercel

1. 打开 Vercel 项目 → **Settings → Environment Variables**
2. 新增一条:
   - Key:`DOUBAO_API_KEY`
   - Value:你刚才复制的那串
   - 三个 Environment 都勾上
3. Save
4. 回 Deployments,**Redeploy**(不勾 cache),等 ✓ Ready

---

## Step 5:测试

打开你的 vercel 网站:

1. 选品牌 → 选产品 → 选文案类型 → 生成文案
2. 文案出来后,点底部"🎨 生成封面图"按钮
3. 在新出现的面板里:
   - 点右上"✨ 一键导入"自动填好 3 个字段(主标题/副标题/花字)
   - 上传一张产品照
   - 点"✨ 生成 3 张候选封面"
4. 等 20-50 秒,会出现 3 张图,点缩略图看大图,点"⬇ 下载"保存

## 文字渲染的现实预期

豆包 SeedEdit 3.0 对中文文字的渲染水平,你心里要有数:

| 字段 | 长度 | 准确率 |
|---|---|---|
| 主标题 | 8-14 字 | 70-85%(偶尔出现错字 / 假汉字) |
| 副标题 | 4-8 字 | 80-90% |
| 花字 | 3-6 字 | 85-95% |

**每张图的"完美率"约 60-70%**。所以系统一次出 3 张就是为了"中彩概率":3 张中至少 1-2 张文字 OK 的概率约 90%。

如果某一类产品 / 某一种文字组合经常出错,**有 2 个解决路径**:

1. **改 prompt 模板**:在 `api/cover-generate.py` 的 `PROMPT_TEMPLATES` 里调整文字描述,如加更多"严格按提供文字一字不差"等强调
2. **走"AI 图 + 程序贴字"混合方案**:让 AI 只负责美化图(不写字),程序用 PIL 精准叠加文字。这套方案我们之前讨论过,如果你最终发现"AI 写字"实在不稳,告诉我,半天可以切换过去

## 故障排查

| 报错 | 原因 | 解决 |
|---|---|---|
| `DOUBAO_API_KEY 未配置` | Vercel 环境变量没设 / 没 Redeploy | 回 Step 4 |
| `豆包 API 错误 401 InvalidApiKey` | Key 写错了或失效 | 重新创建一个 |
| `豆包 API 错误 403 NoPermission` | 模型没开通 | 回 Step 2 第 4 点开通 SeedEdit |
| `豆包 API 错误 429 RateLimit` | 短时间内请求太多 | 等几秒再点生成 |
| `豆包 API 错误 InsufficientBalance` | 账户余额不够 | 火山引擎控制台充值 |
| 3 张都生成失败 | 图片格式问题 / 文字过长 / Prompt 触发了安全审核 | 换张图、缩短文字 |
